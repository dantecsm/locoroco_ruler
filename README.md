# 目标
输入一张图,输出这张图的一笔画方案（动态图）

# 实现原理
先提取特征，将图像数据翻译成一个邻接矩阵
然后用数学方法求解该矩阵的一笔画方案
最后根据生成的方案将图片重构为一笔画动态图输出
